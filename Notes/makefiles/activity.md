Grab my classroom demo code,

```
cd ~/cs50
cp -r ~cs50/demo demo
```

and **work in pairs** to modify the *Makefile* so it adds a phony *test* target, enabling you to later type

```
make test
```

and see it *build names* (if necessary) and *run a test* of the `names` program.
